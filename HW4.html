t = np.linspace(0,np.pi*100,np.int(1e5))
N = 10 + np.sin(t)
bkgd = stats.norm.rvs(size=np.int(1e5))*np.sqrt(N)+N
plt.plot(t,bkgd)
plt.xlabel('t',fontsize=18)
plt.ylabel('Background',fontsize=18)
plt.title('Plot of the Background',fontsize=18)
plt.show

2b) What we are plotting is the number of occurrences of a background measurements in a set time period. This is how we deal with overlapping points in a scatter plot. So the color is the number of point that we would see in a particular square if it was a scatter plot, so a purple square means 0 points there and a yellow square is a square with around 60 points of data
